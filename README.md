# DirectX11 자체 렌더링 엔진
* ### 프로젝트 설명
  DirectX11 기술력 강화를 위한 프로젝트
* ### 담당 업무
  - 툴 제작
  - Material
  - Shadow
  - Mesh Render
  - Data Driven(Json)
  - Asset Import
  - Animation
* ### 사용기술
  - C++
  - DirectX11
  - wxWidgets
  - ASSIMP
